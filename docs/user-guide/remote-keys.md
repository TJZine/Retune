# Remote Control Reference

Retune is designed for the standard LG Magic Remote, but works with basic IR remotes and keyboards (for development) as well.

## Button Mapping

| LG Remote Button | Keyboard Key | Action | Context |
|------------------|--------------|--------|---------|
| **D-Pad (Arrows)** | Arrow Keys | Navigate menus / EPG | Global |
| **OK / Wheel Click** | Enter | Select item / Tune channel | Global |
| **Back** | Backspace / Esc | Close menu / Go back / Show Menu | Global |
| **Guide** | 'G' | Open Electronic Program Guide | Global |
| **Info** | 'I' | Show Program Info overlay | Player |
| **Play / Pause** | Space | Toggle Playback | Player |
| **CH +** | Page Up | Next Channel | Player / EPG |
| **CH -** | Page Down | Previous Channel | Player / EPG |
| **Red Button** | F1 | *(Reserved)* | - |
| **Green Button** | F2 | Open EPG (fallback) | Global |
| **Yellow Button** | F3 | Open Settings | Global |
| **Blue Button** | F4 | *(Reserved)* | - |
| **0-9** | 0-9 | Direct Channel Entry | Player |

## Menu Overlay

While watching video:

1. Press **Back** or **OK** to bring up the UI overlay.
2. The overlay shows:
   - Current program info
   - Progress bar
   - Quick settings buttons
   - Navigate to **Guide** or **Channels**
3. Press **Back** again to hide it.
