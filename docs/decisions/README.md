# Architecture Decision Records (ADR)

This directory contains records of significant architectural decisions made for Retune.

## What is an ADR?

An **Architecture Decision Record** captures a single AD, such as a choice of technology, a pattern, or a significant refactoring. It describes the context, the decision, and the consequences.
